Site Plan: Exploring the Volcanoes of Ecuador

Site Name

Exploring the Volcanoes of Ecuador • This name represents an educational and tourist-focused website highlighting the most iconic volcanoes in Ecuador. The name is specific, geographic, and easy to remember. It helps establish strong branding for both national and international visitors.

Optional domain availability: volcanoes-ecuador.org

Site Purpose

The purpose of this site is to provide educational content and travel inspiration centered around Ecuador’s volcanoes. It will feature facts, geography, natural significance, tourism opportunities, and interactive content such as a dynamic list of volcanoes and a user-friendly form. The site is designed to be responsive, accessible, and informative for students, tourists, and geography enthusiasts.

Scenarios

  • Which volcanoes in Ecuador are currently active and safe to visit?
  • What is the height of Chimborazo, and why is it known as the closest point to the sun?

Color Scheme

The site uses a warm–cool color scheme to visually represent the heat of volcanic activity and the cool mountainous environment:

  • Warm Red (#d62828): Used for headers, call-to-action buttons, and accents.
  • Cool Blue (#3a86ff): Used for links, borders, and icons.
  • Light Gray (#f4f4f4): Used for background and section containers.

Typography

  • Montserrat: Used for headers and titles for a bold, modern look.
  • Open Sans: Used for body text to ensure readability and consistency.

Planned Content Description

The website will consist of at least three pages:

  • Home Page (index.html): General information about Ecuador’s volcanic activity and the significance of volcanoes in the country.
  • Volcanoes Page (volcanoes.html): A dynamic list of volcanoes (e.g., Cotopaxi, Chimborazo, Tungurahua) displayed using JavaScript-powered cards.
  • References Page (references.html): A simple page listing sources for text and images, linked from the footer.

The site will use CSS variables for theming, media queries for responsiveness, and lazy-loading images to improve performance. JavaScript will dynamically generate volcano cards, interact with the DOM, handle forms, use arrays and objects, include conditional logic, and use localStorage for persistent data.
